# rhythmplayer

## Setup
```bash
git clone https://github.com/chanceplus/rhythmplayer.git
```
- Download CMake: https://cmake.org/download/
- 

## Run

### Linux
```bash
./build.sh
./run.sh
```

### Windows
```
./build.bat
./run.bat
```

## Test

### Linux
```bash
./build.sh
./test.sh
```

### Windows
```
./build.bat
./test.bat
```
